Generated Minecraft Fabric Mod Project

IMPORTANT:
This ZIP is a source project, not the finished .jar.

BEST WAY TO BUILD:
1. Unzip this project.
2. Upload all files to a GitHub repository.
3. Go to the repository's Actions tab.
4. Run "Build Minecraft Fabric Mod".
5. Download the generatedmod-jar artifact.
6. Put the finished .jar file into your Minecraft mods folder.

LOCAL BUILD OPTION:
If you have Java and Gradle installed, you can also run:
gradle build

The finished mod .jar will appear in:
build/libs/
